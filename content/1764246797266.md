# Markdown Cheatsheet

## Headers
# H1
## H2
### H3
#### H4
##### H5
###### H6


## Emphasis
**Bold** or __Bold__
*Italic* or _Italic_
***Bold Italic***
~~Strikethrough~~

## Lists
### Unordered
- Item 1
- Item 2
  - Sub Item 2a
  - Sub Item 2b

### Ordered
1. First
2. Second
3. Third

### Task List
- [x] Done task
- [ ] Todo task

## Links & Images
